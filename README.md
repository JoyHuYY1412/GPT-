## Research Pulse

本仓库新增 `research_pulse/` 子目录，里面是一个本地优先的科研信息推送网站：

- arXiv daily、近两年高影响力 paper、论文考古、AI for Science、学术人物关系网。
- 支持管理员审批用户、个人兴趣画像、收藏、收件箱和 Markdown 笔记。
- 支持 DeepSeek 后端问答、复制上下文到 GPT、飞书 webhook 消息提醒。
- 支持从论文 PDF 提取 Figure 1 / Figure 2。

详细启动、`.arxivreader` 虚拟环境、DeepSeek/飞书配置见：

```text
research_pulse/README.md
```
