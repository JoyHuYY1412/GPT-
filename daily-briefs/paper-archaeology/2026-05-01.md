# 考古论文雷达｜2026-05-01

> 目标：每天从 2018–2026 年顶会顶刊或重要技术脉络中，挖几篇想法新奇、跨理论迁移明显、但未必已经成为高引用常识的论文。这个栏目不受当前研究兴趣限制。

## 筛选偏好

- 第一次或较早把某种数学、认知、规划、物理、因果、博弈、信息论、程序化思想引入新任务。
- 强组或强作者的论文，但不一定是大众最熟知的爆款论文。
- 引用量不一定低，但希望关注被主流叙事遮住的技术角度。
- 优先给出论文链接、项目页或代码链接。

## 今日考古条目

## 1. Learning to Shape Rewards using a Game of Two Partners (2020)

- **Venue/类型**: ICLR 2020
- **论文链接**: https://arxiv.org/abs/1908.09817
- **项目/代码**: 链接待补
- **关键词**: reward shaping, multi-agent, RL, planning

### 新奇点

把 reward shaping 看成两个 agent 之间的协作博弈。

### 为什么值得考古

将多智能体协作与 reward design 结合，提供了不同于手工 reward 的视角。

### 为什么可能没被充分关注

现在 RLHF/RLAIF 讨论很多，但早期 reward shaping 的多智能体视角常被跳过。

### 可延展角度

适合连接 agentic RL、reward design、自动反馈生成。

## 2. The Consciousness Prior (2017/idea precursor)

- **Venue/类型**: arXiv / cognitive prior
- **论文链接**: https://arxiv.org/abs/1709.08568
- **项目/代码**: 链接待补
- **关键词**: cognition, representation, attention, interpretability

### 新奇点

把认知科学中的 conscious attention / low-dimensional conscious state 作为 representation learning prior。

### 为什么值得考古

它尝试从认知结构出发，约束高维表示中哪些变量进入可推理的“意识状态”。

### 为什么可能没被充分关注

论文较早且偏概念，但与今天 sparse feature、SAE、interpretable representation 有潜在呼应。

### 可延展角度

适合考古认知理论如何影响 representation learning。

---

## 3. A Simple Neural Attentive Meta-Learner (2018)

- **Venue/类型**: ICLR 2018
- **论文链接**: https://arxiv.org/abs/1707.03141
- **项目/代码**: 链接待补
- **关键词**: meta-learning, memory, attention, in-context learning

### 新奇点

把 memory-augmented attention 用于 meta-learning，让模型通过 attention 检索支持集。

### 为什么值得考古

它在 few-shot learning 中引入了显式记忆检索结构，而不是只靠梯度快速适应。

### 为什么可能没被充分关注

今天大家常讲 in-context learning，却较少回看早期 attention memory meta-learner。

### 可延展角度

适合连接 ICL、memory、retrieval、meta-learning 的思想谱系。

---

## 4. Learning to Compose Task-Specific Tree Structures (2019)

- **Venue/类型**: AAAI 2019
- **论文链接**: https://arxiv.org/abs/1707.02786
- **项目/代码**: 链接待补
- **关键词**: tree, compositionality, reasoning, structure

### 新奇点

把可学习树结构引入任务特定组合推理。

### 为什么值得考古

它不是固定句法树，而是让模型学习对任务有用的组合结构。

### 为什么可能没被充分关注

Transformer 统一 attention 后，很多结构化归纳偏置工作被边缘化，但这类思想仍适合复杂推理。

### 可延展角度

适合考古结构化推理、neural-symbolic、compositionality。

---

## 今日小结

这个栏目更像是 idea mining，而不是热点追踪。建议重点看每篇论文背后的“理论迁移方式”：它把什么外部概念引入了什么 AI 任务，以及这种引入是否还能迁移到今天的大模型、多模态、生成、agent 或科学智能问题中。
