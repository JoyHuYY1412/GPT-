## 1. Matrix-Game 3.0: Real-Time and Streaming Interactive World Model with Long-Horizon Memory

- **arXiv**: https://arxiv.org/abs/2604.08995
- **HuggingFace Papers**: https://huggingface.co/papers/2604.08995
- **方向**: interactive video generation, world model, long-horizon memory

### 它在做什么

Matrix-Game 3.0 关注实时、流式、可交互的世界模型，目标是在高分辨率视频生成中维持长程时空一致性，并支持实时交互。

### 核心技术

- memory-augmented interactive world model
- Video-Pose-Action-Prompt 数据构造
- residual modeling 与 imperfect generated frame re-injection
- camera-aware memory retrieval and injection
- multi-segment autoregressive distillation、量化和 VAE decoder pruning

### 为什么值得关注

它把视频生成从“短片合成”推进到“可交互世界模拟”，对 long-video consistency、world model memory、视频生成评估都很有参考价值。

### 和你相关的角度

可以用于思考：视频生成评估是否应该从画质和文本一致性，进一步转向 memory consistency、interaction stability 和 controllability。

## 2. HY-Embodied-0.5: Embodied Foundation Models for Real-World Agents

- **arXiv**: https://arxiv.org/abs/2604.07430
- **GitHub**: https://github.com/Tencent-Hunyuan/HY-Embodied
- **HuggingFace Papers**: https://huggingface.co/papers/2604.07430
- **方向**: embodied foundation model, VLM, VLA, robot control

### 它在做什么

HY-Embodied-0.5 是面向真实具身 agent 的 foundation model family，覆盖空间/时间视觉感知、具身推理、预测、交互和规划。

### 核心技术

- Mixture-of-Transformers 架构
- latent tokens 增强细粒度视觉感知
- iterative self-evolving post-training
- on-policy distillation
- VLM foundation 到 VLA robot control 的迁移

### 为什么值得关注

这类工作说明 embodied AI 不是简单把 VLM 接到 action head，而是需要面向具身场景重构感知、推理、后训练和蒸馏流程。

### 和你相关的角度

可作为 multimodal foundation model 如何面向具身任务做结构设计和后训练的参考。

## 3. AIM: Intent-Aware Unified World Action Modeling with Spatial Value Maps

- **arXiv**: https://arxiv.org/abs/2604.11135
- **HuggingFace Papers**: https://huggingface.co/papers/2604.11135
- **方向**: world-action model, robot manipulation, spatial value map

### 它在做什么

AIM 关注视频 world model 到机器人控制之间的结构缺口，提出用 spatial value map 作为未来视觉预测与 action generation 之间的显式接口。

### 核心技术

- spatial value map 表示交互位置和操作意图
- intent-causal attention 限制 future information 路由
- self-distillation reinforcement learning
- 在 RoboTwin 2.0 上验证长程和接触敏感任务

### 为什么值得关注

它提出了一个很清晰的问题：未来视频预测不等于可执行控制，必须有面向 action 的中间表示。

### 和你相关的角度

它和你经常强调的“显式表达”“可控性”“验证反馈”非常贴近，也适合迁移到视频编辑 agent 的局部修改规划。

## 4. VAG: Dual-Stream Video-Action Generation for Embodied Data Synthesis

- **arXiv**: https://arxiv.org/abs/2604.09330
- **ResearchTrend**: https://researchtrend.ai/papers/2604.09330
- **方向**: video-action generation, embodied data synthesis, flow matching

### 它在做什么

VAG 试图同步生成视频和动作轨迹，用于具身数据合成，解决 world model 生成视频但缺少 paired action trajectory 的问题。

### 核心技术

- dual-stream video-action generation
- flow matching
- synchronized denoising
- adaptive 3D pooling 传递视频上下文到 action branch

### 为什么值得关注

它和 AIM 可以放在一起看：AIM 强调 spatial value map，VAG 强调 video-action joint generation。二者都在回答“视频生成如何成为可执行行动模型”。

### 和你相关的角度

如果把 action 替换成镜头动作、剪辑动作、脚本动作，这种 video-action joint generation 或许可以启发短视频创作 agent。

## 今日趋势总结

今天的几篇工作共同指向一个趋势：**video model 正在被重新定义为 world/action model**。值得继续追的问题包括：world model 需要什么形式的 memory，视频预测如何转成 action，生成模型如何产生可执行数据，以及 video generation evaluation 是否要加入交互一致性和行动可用性指标。
