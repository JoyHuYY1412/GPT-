## 1. Genie: Generative Interactive Environments

- **年份/类型**: 2024, ICML 2024
- **论文链接**: https://arxiv.org/abs/2402.15391
- **项目页**: https://sites.google.com/view/genie-2024/
- **关键词**: world model, interactive environment, action-conditioned video generation

### 为什么和当前研究相关

Genie 试图从无标注视频中学习可交互环境，把视频生成和 action-conditioned dynamics 连接起来。它不是单纯生成视频，而是将视频作为可操作环境建模。

### 可以延展成自己的工作角度

- 用于 framing “video generation as interactive simulator”。
- 讨论视频生成评估中 interaction consistency 的必要性。
- 对短视频 agent 来说，可以类比为“创意动作”控制下的视频状态转移。

## 2. VideoPoet: A Large Language Model for Zero-Shot Video Generation

- **年份/类型**: 2023/2024, ICML 2024
- **论文链接**: https://arxiv.org/abs/2312.14125
- **项目页**: https://sites.research.google/videopoet/
- **关键词**: autoregressive video generation, visual tokenization, multimodal generation

### 为什么和当前研究相关

VideoPoet 把视频生成表述成语言模型式 token prediction，是理解 AR video generation 和视觉 tokenizer 路线的重要节点。

### 可以延展成自己的工作角度

- 和 JPEG-LM、VideoFlexTok、UniTok、xAR 等视觉 tokenization 工作形成谱系。
- 思考视觉 token 表示是否能同时服务生成、理解和编辑。

## 3. GAIA: a Benchmark for General AI Assistants

- **年份/类型**: 2023/2024, ICLR 2024
- **论文链接**: https://arxiv.org/abs/2311.12983
- **Benchmark**: https://huggingface.co/gaia-benchmark
- **关键词**: agent, tool use, benchmark, evaluation

### 为什么和当前研究相关

GAIA 关注通用 AI assistant 在搜索、工具使用、多步骤推理中的能力。它适合作为设计 agentic benchmark 的参照，而不只是普通问答 benchmark。

### 可以延展成自己的工作角度

- 给多模态 agent 或短视频创作 agent 设计端到端评估。
- 将“是否能完成任务”而不是“回答是否像样”作为核心指标。

## 4. Visual Program Distillation

- **年份/类型**: 2023, CVPR 2023
- **论文链接**: https://arxiv.org/abs/2212.03052
- **项目页**: https://prior.allenai.org/projects/visprog
- **关键词**: visual reasoning, program, tool use, agent

### 为什么和当前研究相关

这篇工作把视觉任务拆成程序化工具调用，是 agentic visual reasoning 的早期重要路线。它对你现在讨论的验证反馈、多工具短视频创作 agent、显式中间结构都有启发。

### 可以延展成自己的工作角度

- 将复杂视频生成任务拆成脚本规划、素材检索、局部生成、验证反馈等可解释程序。
- 让 agent 的工具调用轨迹成为可评估对象。

## 5. Segment Anything

- **年份/类型**: 2023, ICCV 2023
- **论文链接**: https://arxiv.org/abs/2304.02643
- **项目页**: https://segment-anything.com/
- **GitHub**: https://github.com/facebookresearch/segment-anything
- **关键词**: promptable perception, mask, foundation model

### 为什么和当前研究相关

SAM 是 promptable visual foundation model 的代表。它的意义不仅是分割性能，而是将 mask 作为可交互、可工具化、可迁移的视觉中间层。

### 可以延展成自己的工作角度

- 连接 MTA-CLIP、mask-text alignment 和 text-enhanced segmentation。
- 在视频编辑/短视频 agent 中作为主体一致性、局部编辑和质量验证工具。
