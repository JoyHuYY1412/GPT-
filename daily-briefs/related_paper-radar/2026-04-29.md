## 1. PaLM-E: An Embodied Multimodal Language Model

- **年份/类型**: 2023, ICML 2023
- **论文链接**: https://arxiv.org/abs/2303.03378
- **项目页**: https://palm-e.github.io/
- **匹配标签**: embodied, multimodal, robot, language model

### 为什么和当前研究相关

PaLM-E 把多模态感知、语言模型和具身控制连接起来，是 embodied multimodal model 的代表节点。它不是只做视觉问答，而是尝试让语言模型接收机器人传感输入并参与真实控制。

### 可以延展成自己的工作角度

适合对比 LMM、VLA、world model 三条路线如何连接感知、语言和行动，也适合放入多模态 Agent 的 related work backbone。

---

## 2. VIMA: General Robot Manipulation with Multimodal Prompts

- **年份/类型**: 2022/2023, ICML 2023
- **论文链接**: https://arxiv.org/abs/2210.03094
- **项目页**: https://vimalabs.github.io/
- **匹配标签**: robot, multimodal prompt, manipulation, generalist policy

### 为什么和当前研究相关

VIMA 将多模态 prompt 引入机器人操作任务，强调任务描述、视觉目标和动作执行之间的统一接口。它把 prompt 从语言扩展到视觉和动作条件，是 promptable embodied policy 的重要例子。

### 可以延展成自己的工作角度

可用于思考多模态任务条件化：在短视频创作 Agent 中，商品图、历史素材、脚本和目标人群是否也能组成 multimodal prompt。

---

## 3. RT-2: Vision-Language-Action Models Transfer Web Knowledge to Robotic Control

- **年份/类型**: 2023, CoRL 2023 / robotics foundation model
- **论文链接**: https://arxiv.org/abs/2307.15818
- **项目页**: https://robotics-transformer2.github.io/
- **匹配标签**: VLA, robot, web knowledge, action

### 为什么和当前研究相关

RT-2 把 web-scale VLM 知识迁移到机器人动作输出，推动了 vision-language-action model 的主流叙事。它把自然语言、视觉理解和动作生成放进同一个模型接口。

### 可以延展成自己的工作角度

适合和 HY-Embodied、Open X-Embodiment、world-action model 放在一条谱系里，也能帮助区分 VLA 和 video world model 的不同建模目标。

---

## 4. Do As I Can, Not As I Say: Grounding Language in Robotic Affordances

- **年份/类型**: 2022, CoRL 2022
- **论文链接**: https://arxiv.org/abs/2204.01691
- **项目页**: https://say-can.github.io/
- **匹配标签**: robot, affordance, planning, language grounding

### 为什么和当前研究相关

SayCan 将语言模型规划与 affordance/value function 结合，解决“会说但不能做”的具身落地问题。它的关键不在于 LLM 有多强，而在于用可执行性约束过滤语言计划。

### 可以延展成自己的工作角度

这对多模态 Agent 的验证反馈非常有启发：生成脚本、镜头计划或工具调用计划之后，需要一个 affordance-like verifier 判断是否真的可执行。

---

## 5. Ego4D: Around the World in 3,000 Hours of Egocentric Video

- **年份/类型**: 2022, CVPR 2022
- **论文链接**: https://arxiv.org/abs/2110.07058
- **项目页**: https://ego4d-data.org/
- **匹配标签**: egocentric video, long video, benchmark, dataset

### 为什么和当前研究相关

Ego4D 提供大规模第一人称长视频数据和任务设置，是长视频理解、具身感知和日常活动建模的重要数据节点。它把 video understanding 从短片分类推进到长期记忆、预测和人类活动理解。

### 可以延展成自己的工作角度

适合作为 long-video reasoning、agent perception、memory benchmark 的背景数据工作，也适合和 GameplayQA、VideoMME、EgoSchema 等视频评估工作形成谱系。

---

## 6. DreamerV3: Mastering Diverse Domains through World Models

- **年份/类型**: 2023/2025, world model RL line
- **论文链接**: https://arxiv.org/abs/2301.04104
- **项目页**: https://danijar.com/project/dreamerv3/
- **匹配标签**: world model, RL, planning, latent dynamics

### 为什么和当前研究相关

DreamerV3 展示了 latent world model 在多任务 RL 中的通用性，是理解 world model 与 policy learning 结合的重要节点。它强调从 latent dynamics 中学习策略，而不是直接依赖像素空间预测。

### 可以延展成自己的工作角度

适合对比 video world model 与 RL latent world model 的差别：前者强调可视化生成和交互一致性，后者强调可用于规划和控制的抽象状态。

---

## 7. Open X-Embodiment: Robotic Learning Datasets and RT-X Models

- **年份/类型**: 2023/2024, ICRA 2024 / robotics data scaling line
- **论文链接**: https://arxiv.org/abs/2310.08864
- **项目页**: https://robotics-transformer-x.github.io/
- **匹配标签**: robot, dataset, scaling, cross-embodiment

### 为什么和当前研究相关

Open X-Embodiment 强调跨机器人、跨任务数据汇聚，是 embodied foundation model 的数据基础设施节点。它说明具身智能的扩展不仅靠模型，也依赖跨平台数据标准化。

### 可以延展成自己的工作角度

适合比较多模态数据融合、跨域泛化和具身大模型训练范式，也能启发你关于多源异构多智能体系统的数据组织问题。

## 8. RoboCat: A Self-Improving Robotic Agent

- **年份/类型**: 2023, robotics agent / self-improving agent line
- **论文链接**: https://arxiv.org/abs/2306.11706
- **项目页**: https://www.deepmind.com/blog/robocat-a-self-improving-robotic-agent
- **匹配标签**: robot, agent, self-improving, embodied

### 为什么和当前研究相关

RoboCat 强调机器人 agent 通过数据闭环和自我改进扩展任务能力。它将模型、数据采集、任务泛化和持续改进放入一个循环。

### 可以延展成自己的工作角度

可连接 agentic RL、反馈闭环和具身数据生成，也可类比到短视频创作 Agent：生成候选、验证反馈、再生成和策略更新形成闭环。
