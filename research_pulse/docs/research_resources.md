# Research Pulse 资料源

## 论文发现

- [Hugging Face Daily Papers](https://huggingface.co/papers)：社区热度、每日/每周/月度 paper 信号。
- [CodeSOTA Trending](https://www.codesota.com/papers-with-code/trending)：Papers with Code 趋势页替代，可看 code、benchmark、score。
- [Papers with Code 2](https://paperswithcode2.com/)：paper、代码、数据集和 benchmark 聚合。

## 论文阅读与 PDF 问答

- [paper-qa](https://github.com/whitead/paper-qa)：文档问答、引用和 RAG 式 paper 阅读的开源参考。
- [Zotero Better Notes](https://github.com/windingwind/zotero-better-notes)：Zotero 内 Markdown 笔记和论文标注工作流。
- [Zotero Better BibTeX](https://github.com/retorquere/zotero-better-bibtex)：Zotero 到 Markdown/LaTeX 写作的引用键和 bib 管理。

## 笔记与飞书

- Research Pulse 笔记默认保存为 Markdown，并在网页内渲染标题、加粗、列表、代码、链接、公式样式和图片。
- 飞书 webhook 能发送消息；自动创建飞书文档需要配置飞书开放平台应用凭证与文档 API 权限。
- 飞书文档导入接口参考：[创建导入任务](https://open.feishu.cn/document/server-docs/docs/drive-v1/import_task/create)。
